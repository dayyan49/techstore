TechStore is a modern frontend e-commerce store built with HTML, CSS, and JavaScript. It features product filtering, a shopping cart, category filters, and a responsive design.

Features:
  i)Browse products by category(Audio, Wearables, Computers, Mobile).
  ii)Real-time product search.
  iii)Add to cart with quantity management.
  iv)Responsive UI for mobile and desktop.

Technologies Used:
  i)HTML5
  ii)CSS3
  iii)JavaScript
